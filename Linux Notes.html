Setting up Version Control

  • Initialize local repository
    • Cmd: git init
    • Cmd: git clone {repository-url}
  • Make commitable edits and commit
    • Cmd: git add {file}
    • Cmd: git commit -m "{Commit message}"
  • Identify author information if needed
    • Cmd: git config --global user.name "{Your Name}"
    • Cmd: git config --global user.email "{Your Email}"
  • Create a remote repo on GitHub and push (requires GitHub CLI)
    • Cmd: gh repo create {repo-name} --public --source=. --remote=origin --push
    • Cmd: git push --set-upstream origin master

Viewing Files

  • Cmd: ls - List files in the current directory
  • Cmd: cat {file} - Display the contents of a file
  • Cmd: head {file} -n 3 - Show the first 3 lines of a file
  • Cmd: less {file} - View the contents of a file one screen at a time

Editing Files

  • Cmd: nano {file} - Edit a file using the nano text editor
  • Cmd: vim {file} - Edit a file using the vim text editor
  • Cmd: code {file} - Edit a file using Visual Studio Code

Directory Commands

  • Cmd: cd {directory path}: Change directory
  • Cmd: cd -: Change to the previous directory
  • Cmd: pwd: Print working directory
  • Cmd: ls -l: List files with detailed information
  • Cmd: ls -a: List all files, including hidden files

File Management

  • Cmd: touch {file} - Create a new empty file
  • Cmd: cp {source} {destination} - Copy a file
  • Cmd: mv {source} {destination} - Move or rename a file
  • Cmd: rm {file} - Remove a file
  • Cmd: mkdir {directory} - Create a new directory
  • Cmd: rmdir {directory} - Remove an empty directory
  • Cmd: rm -r {directory} - Remove a directory and its contents
